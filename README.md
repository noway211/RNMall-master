## 实现功能
* 基于已有的Android工程集成React-Native代码，
* RN实现的一个标准的商城类App首页，仿京东,天猫
* 流畅的实现了自定义下拉刷新动画和载入动画,以及网络异常界面处理
* 优化了RN框架启动时白屏的问题
